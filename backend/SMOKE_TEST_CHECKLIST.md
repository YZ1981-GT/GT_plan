# 发版前冒烟测试清单

每次发版前必须执行以下检查，全部通过后方可发布。

## 一、后端测试

```bash
cd backend
$env:PYTHONPATH = "."
python -m pytest tests/ --ignore=tests/test_collaboration.py -q --tb=short
```

预期：全部通过，无 FAILED。

## 二、主链路冒烟（手动）

### 2.1 登录链路
- [ ] 打开 http://localhost:3030/login
- [ ] 输入 admin/admin123 登录
- [ ] 验证跳转到首页，左侧导航正常

### 2.2 项目列表
- [ ] 项目列表只显示当前用户有权限的项目（非 admin 用户验证）
- [ ] 创建新项目，填写基本信息，确认成功

### 2.3 底稿主链路
- [ ] 进入项目 → 底稿列表 → 选择底稿
- [ ] 点击"下载编辑" → 文件下载成功
- [ ] 点击"在线编辑" → Univer 编辑器正常加载，显示底稿内容
- [ ] 上传编辑后的底稿 → 版本号递增
- [ ] 版本冲突时显示冲突提示

### 2.4 复核提交
- [ ] 未分配复核人时，提交复核被阻断（显示原因）
- [ ] 分配复核人 + 执行 QC 自检 → 通过后可提交
- [ ] 有未解决批注时，提交被阻断

### 2.5 附件链路
- [ ] 上传附件 → 列表显示
- [ ] 点击预览 → PDF/图片可预览，其他显示 OCR 文本
- [ ] 点击下载 → 文件下载成功
- [ ] 关联底稿 → 搜索下拉选择底稿 → 关联成功

### 2.6 试算表
- [ ] 查看试算表 → 数据正常显示
- [ ] 合并锁定期间 → 重算按钮返回 423

## 三、关键 API 验证

```bash
# 健康检查
curl http://localhost:9980/api/health

# 版本信息
curl http://localhost:9980/api/version

# 功能开关
curl http://localhost:9980/api/feature-flags

# 任务中心
curl http://localhost:9980/api/tasks/stats
```
