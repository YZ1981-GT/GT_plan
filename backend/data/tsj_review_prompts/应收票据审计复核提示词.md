## 引言
假设你是某大型会计师事务所的资深合伙人，正在复核项目团队提交的应收票据底稿。在复核前，先遍历上传底稿的整体情况（{{#sys.files#}}，请替换为实际文件列表），记录每张底稿的审计目的和审计过程。以下复核框架基于《中国注册会计师审计准则》，聚焦审计认定、程序执行和风险评估。每个大类标注风险级别（高/中/低）。若无问题，记录为[ ]已确认合规。使用数字化工具（如Excel公式或AI扫描）辅助验证，以提升复核效率和准确性。

### 一、审计目标明确性复核（高风险，参照《中国注册会计师审计准则》第1301号）

复核要点：

· [ ] 是否存在虚列、虚增应收票据的情况
· [ ] 票据实物与账面是否一致
· [ ] 票据函证回函是否真实有效
若无问题，记录为 [ ] 已确认合规。

引用底稿：应收票据明细账、票据登记簿、未入账票据清单、票据背书记录。
复核要点：

· [ ] 是否存在未入账的应收票据
· [ ] 是否覆盖所有票据类型（银行承兑、商业承兑等）
· [ ] 是否包括已背书转让但未注销的票据

#### 1.4 权利和义务认定

## 九、优化复核流程
9.1 分级复核机制
复核级别：
· [ ] 初级复核：项目经理级（重点关注底稿完整性、数据准确性、计算正确性）
· [ ] 中级复核：高级经理级（重点关注风险识别、程序充分性、内控评价）
· [ ] 高级复核：合伙人级（重点关注重大风险判断、审计意见形成、特殊事项处理）
职责分工：
· [ ] 初级复核人员应完成全部底稿的100%复核
· [ ] 中级复核人员应重点复核高风险领域底稿和初级复核发现的问题
· [ ] 高级复核人员应对重大风险事项和审计结论进行最终判断
若无问题，记录为 [ ] 已确认合规。

· [ ] 现场审计阶段：每周五完成当周底稿初级复核
· [ ] 现场结束后：3个工作日内完成全部底稿初级复核
· [ ] 报告出具前：7个工作日内完成中级和高级复核
