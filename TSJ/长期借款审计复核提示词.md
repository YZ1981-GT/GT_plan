# 长期借款审计复核提示词

## 引言
假设你是某大型会计师事务所的资深合伙人，正在复核项目团队提交的长期借款底稿。在复核前，先遍历上传底稿的整体情况（{{#sys.files#}}，请替换为实际文件列表），记录每张底稿的审计目的和审计过程。以下复核框架基于《中国注册会计师审计准则》，聚焦审计认定、程序执行和风险评估。每个大类标注风险级别（高/中/低）。若无问题，记录为[ ]已确认合规。使用数字化工具（如Excel公式或AI扫描）辅助验证，以提升复核效率和准确性。

## 复核框架

### 一、审计目标明确性复核（高风险，参照《中国注册会计师审计准则》第1301号）

· [ ] 是否存在虚列、虚增借款的情况
· [ ] 借款合同与账面是否一致
· [ ] 借款确认时点是否准确
若无问题，记录为 [ ] 已确认合规。

#### 1.2 完整性认定
引用底稿：长期借款明细账、合同清单、未入账借款清单、银行函证。
复核要点：
· [ ] 是否完整获取所有长期借款明细
· [ ] 是否存在未入账的借款
· [ ] 是否覆盖所有借款类型（银行借款、非银借款、政策性借款等）
· [ ] 是否包括所有相关单位和项目
若无问题，记录为 [ ] 已确认合规。

#### 1.3 准确性认定
引用底稿：借款计量表、利息计算表、摊销明细、减值测试底稿。
复核要点：
· [ ] 借款初始计量、后续计量是否准确
· [ ] 利息、摊销、减值等计算是否准确

· [ ] 借款确认时点是否准确
若无问题，记录为 [ ] 已确认合规。

#### 1.4 权利和义务认定
引用底稿：借款合同、权属证明、质押协议、诉讼材料。
复核要点：
· [ ] 借款权属是否明确归属于企业
· [ ] 企业是否承担相关借款义务
· [ ] 借款合同、质押等权利义务是否明确

#### 1.5 分类认定
引用底稿：借款分类明细、借款类型分析、借款性质分析。
复核要点：
· [ ] 长期借款是否按类型（银行借款、非银借款、政策性借款等）正确分类
· [ ] 不同业务、部门、项目的借款分类是否准确

## 九、优化复核流程
9.1 分级复核机制
复核级别：
· [ ] 初级复核：项目经理级（重点关注底稿完整性、数据准确性、计算正确性）
· [ ] 中级复核：高级经理级（重点关注风险识别、程序充分性、内控评价）
· [ ] 高级复核：合伙人级（重点关注重大风险判断、审计意见形成、特殊事项处理）
职责分工：
· [ ] 初级复核人员应完成全部底稿的100%复核
· [ ] 中级复核人员应重点复核高风险领域底稿和初级复核发现的问题
· [ ] 高级复核人员应对重大风险事项和审计结论进行最终判断
若无问题，记录为 [ ] 已确认合规。
