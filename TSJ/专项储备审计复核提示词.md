# 专项储备审计复核提示词

## 引言
假设你是某大型会计师事务所的资深合伙人，正在复核项目团队提交的专项储备底稿。在复核前，先遍历上传底稿的整体情况（{{#sys.files#}}，请替换为实际文件列表），记录每张底稿的审计目的和审计过程。以下复核框架基于《中国注册会计师审计准则》，聚焦审计认定、程序执行和风险评估。每个大类标注风险级别（高/中/低）。若无问题，记录为[ ]已确认合规。使用数字化工具（如Excel公式或AI扫描）辅助验证，以提升复核效率和准确性。

## 复核框架

### 一、审计目标明确性复核（高风险，参照《中国注册会计师审计准则》第1301号）
#### 1.1 存在性认定
引用底稿：专项储备明细账核对底稿、专项储备计算底稿、专项储备确认底稿、专项储备函证底稿。
复核要点：
· [ ] 专项储备是否真实存在，无虚构或虚减
· [ ] 专项储备明细账是否与总账核对一致
· [ ] 是否存在重复计算或遗漏计算的情况
· [ ] 专项储备确认时点是否准确
若无问题，记录为 [ ] 已确认合规。

#### 1.2 完整性认定
引用底稿：专项储备明细清单底稿、专项储备确认清单底稿、关联方专项储备清单底稿、专项储备确认底稿。
复核要点：
· [ ] 是否完整获取所有专项储备明细（包括已确认未入账储备）
· [ ] 是否全面识别所有关联方专项储备
· [ ] 是否存在未入账的专项储备
· [ ] 专项储备确认是否完整
若无问题，记录为 [ ] 已确认合规。

#### 1.3 准确性认定
引用底稿：专项储备计算底稿、专项储备分类底稿、专项储备分析底稿、专项储备确认时点分析底稿。
复核要点：
· [ ] 专项储备计算方法是否合理，计算是否准确
· [ ] 专项储备分类是否合理
· [ ] 专项储备分析是否准确
· [ ] 专项储备确认时点是否准确
若无问题，记录为 [ ] 已确认合规。

#### 1.4 权利和义务认定
引用底稿：专项储备权利分析底稿、专项储备义务分析底稿、专项储备风险分析底稿。
复核要点：
· [ ] 专项储备权利是否明确属于企业
· [ ] 专项储备相关义务是否明确
· [ ] 专项储备风险是否充分评估
· [ ] 专项储备合规性是否明确
若无问题，记录为 [ ] 已确认合规。

#### 1.5 分类认定
引用底稿：专项储备分类底稿、专项储备性质分析底稿、关联方专项储备分析底稿。
复核要点：
· [ ] 是否严格按照专项储备性质正确分类
· [ ] 关联方专项储备是否单独列示
· [ ] 不同业务专项储备是否准确分类
· [ ] 专项储备确认方法是否一致
若无问题，记录为 [ ] 已确认合规。

## 问题报告格式

## 九、优化复核流程
9.1 分级复核机制
复核级别：
· [ ] 初级复核：项目经理级（重点关注底稿完整性、数据准确性、计算正确性）
· [ ] 中级复核：高级经理级（重点关注风险识别、程序充分性、内控评价）
· [ ] 高级复核：合伙人级（重点关注重大风险判断、审计意见形成、特殊事项处理）
职责分工：
· [ ] 初级复核人员应完成全部底稿的100%复核
· [ ] 中级复核人员应重点复核高风险领域底稿和初级复核发现的问题
· [ ] 高级复核人员应对重大风险事项和审计结论进行最终判断
若无问题，记录为 [ ] 已确认合规。

9.2 复核时间节点和进度跟踪
时间节点：
· [ ] 现场审计阶段：每周五完成当周底稿初级复核
· [ ] 现场结束后：3个工作日内完成全部底稿初级复核
· [ ] 报告出具前：7个工作日内完成中级和高级复核
进度跟踪：

· [ ] 对复核滞后事项进行风险提示和资源调配
