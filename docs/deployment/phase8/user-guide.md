# Phase 8 用户手册

## 概述

Phase 8 聚焦于性能提升和用户体验优化，主要改进包括：

1. **查询性能优化** — 穿透查询、四表联查速度提升
2. **底稿编辑体验** — 虚拟滚动、批量预填、离线缓存
3. **报表导出优化** — Word/PDF 导出速度提升
4. **移动端适配** — 响应式布局、移动端底稿查看
5. **数据校验** — 一致性/完整性自动校验
6. **安全增强** — 数据加密、审计日志增强

---

## 1. 穿透查询优化

### 游标分页

大数据量场景下，穿透查询采用游标分页，支持无限滚动：

- 每次加载 100 条记录
- 滚动到底部自动加载下一页
- 支持虚拟滚动，流畅浏览 10 万+ 条记录

### 使用方式

1. 在试算表中点击科目金额
2. 穿透到明细账/辅助账
3. 列表自动分页加载，无需手动翻页

---

## 2. 底稿编辑优化

### 虚拟滚动

底稿列表支持 1000+ 底稿流畅浏览：
- 只渲染可见区域的底稿项
- 快速滚动无卡顿
- 搜索使用防抖优化

### 批量预填

支持同时预填多个底稿：
1. 选择需要预填的底稿（最多 10 个）
2. 点击"批量预填"按钮
3. 系统并发处理，显示进度

### 离线缓存

已查看的底稿支持离线访问：
- 自动缓存最近查看的底稿
- 断网时可查看已缓存底稿（只读）
- 恢复网络后自动同步

---

## 3. 报表导出

### Word 导出

- 模板缓存：重复导出同类报表更快
- 格式一致：自动校验致同规范（字体/页边距/表格）
- 导出预览：导出前可预览效果

### PDF 导出

- 异步导出：大文档不阻塞操作
- 进度显示：导出面板显示实时进度
- 任务管理：可在任务中心查看导出状态

---

## 4. 移动端适配

### 响应式布局

- 手机端（< 768px）：单栏布局
- 平板端（768px - 1024px）：双栏布局
- 桌面端（> 1024px）：三栏布局

### 移动端底稿

- 简化的单元格编辑器
- 支持下载/上传底稿
- 触摸手势支持（滑动/缩放）

---

## 5. 数据校验

### 自动校验

系统自动检查以下数据一致性：
- 余额表与辅助表金额一致
- 报表与附注数据一致
- 底稿与试算表数据一致
- 调整分录与报表一致

### 校验面板

1. 进入项目 → 数据校验面板
2. 按类型/严重度筛选问题
3. 点击"一键修复"处理常见问题
4. 导出校验报告

### 严重度说明

| 级别 | 说明 | 处理方式 |
|------|------|---------|
| 高 | 阻断性问题 | 必须修复后才能签字 |
| 中 | 需关注问题 | 建议修复 |
| 低 | 提示性问题 | 可选修复 |

---

## 6. 性能监控

### 性能面板（管理员）

管理员可在"系统管理 → 性能监控"查看：
- API 平均响应时间
- 数据库查询时间
- 缓存命中率
- 性能趋势图表

### 性能告警

系统自动监控并告警：
- API 响应超过 3 秒
- 数据库查询超过 1 秒
- 缓存命中率低于 50%

## 7. 安全增强

### 数据加密

- 敏感数据（API 密钥等）加密存储
- 密码使用 bcrypt（cost=14）加密

### 登录保护

- 连续 5 次登录失败自动锁定账户
- 锁定时长 30 分钟
- 异常 IP 自动检测

### 审计日志

- 所有操作记录详细上下文
- 支持导出审计日志（CSV/Excel）
- 异常操作自动告警

## 常见问题

### Q: 穿透查询加载慢？
A: Phase 8 已优化查询性能，如仍然较慢请检查网络连接。系统会自动缓存查询结果（5 分钟有效期）。

### Q: 离线缓存占用多少空间？
A: 离线缓存使用浏览器 localStorage，默认限制约 5MB。系统会自动清理最旧的缓存。

### Q: 如何清除离线缓存？
A: 浏览器设置 → 清除网站数据，或在底稿列表中点击"清除缓存"按钮。

### Q: 批量预填失败怎么办？
A: 部分底稿预填失败不影响其他底稿。失败的底稿会显示错误原因，可单独重试。
