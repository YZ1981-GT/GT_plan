## 二、特殊循环映射核验（M 权益 / 关联方）

### 2.1 M 权益循环（无 B23/C 配对）

**事实**: 权益循环（M1-M10）在 D-N 中独立存在，但 **B23 系列没有 B23-XX 权益**、**C 系列没有 CXX 权益**

**原因**: 致同方法论中权益审计走"完成阶段 + 与治理层沟通"路径（A 类底稿），不走"业务循环控制测试"路径
- A1 财务报告 / A10 与治理层沟通 / M1 实收资本 / M5 盈余公积 / M6 本年利润 直接做实质性程序
- 没有"权益业务循环"概念（权益不是日常业务交易，而是公司决策事项）

**影响**: D 销售循环 spec 不涉及 M 循环联动，本核验仅作记录

### 2.2 关联方及交易循环（独立 B23-14/C15 但无独立 D-N）

**事实**: B23-14 / C15 / C15-2 三个底稿覆盖关联方循环，但**关联方相关的实质性程序散落在各业务循环底稿**:
- D2-6 关联方及交易检查表（D 销售循环内）
- D4-21 关联方销售情况及价格分析（D 销售循环内）
- D6-5 关联关系及交易检查（D 销售循环 D6 合同资产）
- D7-6 关联方关系及交易检查表（D 销售循环 D7 合同负债）
- A7 关联交易（完成阶段独立底稿）
- L1-XX 关联方借款（L 债务循环内）

**结论**: 关联方循环采用"控制层独立 + 实质层散落"模式，不需要独立 D-N 目录。

### 2.3 在建工程/研发/租赁三循环合并方式

**实测**: 模板根目录下 `4.风险应对-实质性程序（D-N）` 没有独立的"在建工程/研发/租赁"目录，但 B23-6/B23-8/B23-13 三个 B 类和 C7/C9/C14 三个 C 类**独立存在**

**推测合并方式**:
- **在建工程**（B23-6/C7）→ 实质性程序合并入 **H 固定资产循环**（在建工程 1604 与固定资产 1601 同属长期资产）
- **研发循环**（B23-8/C9）→ 实质性程序合并入 **K 管理循环**（研发支出 5301 在管理费用类目下，或独立 K 子科目）
- **租赁循环**（B23-13/C14）→ 实质性程序合并入 **H 固定资产循环**（使用权资产 H8）+ **L 债务循环**（租赁负债 L7/L8）

**核验需求**: 这 3 个循环的"控制层独立 + 实质层合并"是否真的覆盖了所有相关科目？需要 §三 详查。

## 四、覆盖核验脚本（用完即删，可重跑）

为快速核验 §三 的 5 处疑问，建议执行以下 grep 脚本:

```python
# scripts/_verify_b23_c_coverage.py（用完即删）
import openpyxl, json, os
from pathlib import Path

ROOT = Path(r"致同通用审计程序及底稿模板（2025年修订）\1.致同审计程序及底稿模板（2025年）")
TARGETS = [
    # 在建工程
    ("H 固定资产循环", ["在建工程", "1604", "工程款", "竣工验收"]),
    # 研发
    ("K 管理循环", ["研发", "5301", "资本化", "费用化"]),
    ("I 无形资产循环", ["研发", "5301", "资本化"]),
    # 租赁
    ("L 债务循环", ["租赁", "使用权资产", "租赁负债", "L7", "L8"]),
    ("H 固定资产循环", ["租赁", "使用权资产", "H8"]),
    # 关联方分布在各循环
    ("F 存货循环", ["关联方"]),
    ("G 投资循环", ["关联方"]),
    ("H 固定资产循环", ["关联方"]),
    ("K 管理循环", ["关联方"]),
    ("L 债务循环", ["关联方"]),
]

results = {}
for cycle, keywords in TARGETS:
    cycle_dir = ROOT / "4.风险应对-实质性程序（D-N）" / cycle
    if not cycle_dir.exists():
        results[(cycle, tuple(keywords))] = "MISSING_DIR"
        continue
    matched_files = []
    for xlsx in cycle_dir.rglob("*.xlsx"):
        for kw in keywords:
            if kw in xlsx.name:
                matched_files.append(xlsx.name)
                break
        else:
            # 也扫 sheet 名
            try:
                wb = openpyxl.load_workbook(xlsx, read_only=True)
                if any(kw in s for kw in keywords for s in wb.sheetnames):
                    matched_files.append(f"{xlsx.name}（sheet 名命中）")
                wb.close()
            except: pass
    results[(cycle, tuple(keywords))] = matched_files

for (cycle, kws), files in results.items():
    print(f"{cycle} × {kws}: {len(files) if isinstance(files, list) else files}")
    if isinstance(files, list):
        for f in files[:5]:
            print(f"  - {f}")
```

**预期结果**: 如果某 cycle × keyword 命中 0 个文件，说明覆盖缺口需补；如果命中 1+ 文件，说明覆盖完整只是散落。
