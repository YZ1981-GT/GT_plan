## 9. 风险与应对

| 风险 | 应对 |
|------|------|
| python-docx 三线表边框精度不够 | 用 OxmlElement 直接操作 XML |
| 仿宋_GB2312 在 Docker 中不存在 | Dockerfile 安装字体包或 COPY 字体文件 |
| 页码域代码在 ONLYOFFICE 中不渲染 | 方案B 模板自带页码，不依赖代码生成 |
| 权益变动表宽表列数过多 | 自动缩小字号到 9pt，或横向页面 |
| 附注章节过多导致导出超时 | 分批导出+进度条，单章节超时跳过 |
| 上年 PDF 报告 OCR 质量差 | MinerU GPU 加速+质量检测+人工确认 |
| LLM 预填内容不准确 | 所有 LLM 内容标记 [需确认]，人工逐处审阅 |
| 集团子企业会计政策差异大 | 差异标记+绿色高亮提示，不自动替换政策段落 |
| 上年附注表格结构与当期不一致 | 结构对比→不一致章节标记为"需手动处理" |
