# B23 销售循环业务层面控制底稿模板库

## 文档概述

本文档为致同会计师事务所销售循环业务层面控制底稿模板库，包含了解、评价并测试销售循环业务层面控制的所有相关底稿模板，适用于致同会计师事务所的审计工作。

## 使用说明

1. **控制了解**：根据被审计单位的实际情况了解销售循环内部控制
2. **控制评价**：按照底稿中的控制评价程序逐步执行
3. **控制测试**：确保所有控制测试程序都有完整的执行记录
4. **结果汇总**：及时汇总控制了解和测试结果

## 📚 底稿间关系说明

### 底稿执行顺序
1. **B23-1** → 总体程序表，指导整个审计过程
2. **B23-1-1** → 整体控制汇总，了解控制环境
3. **B23-1-2** → 流程图及描述，了解业务流程
4. **B23-1-3** → 控制矩阵，识别和评价控制点
5. **B23-1-4** → 穿行测试，验证控制设计
6. **B23-1-5** → 控制测试，执行控制测试
7. **B23-1-6** → 控制缺陷汇总，评价缺陷严重程度
8. **B23-2** → 控制测试汇总，测试控制运行
9. **B23-3** → 控制评价报告，形成最终结论

##### 3.2 销售订单

1、业务人员在全面了解经销商资信情况下，按照GSP要求填写《开户申请表》，选择信用良好的客户进行开户。填写完后，销售部业务员将申请表传真回来报销售经理审核。销售部经理根据客户的以往信用记录，授予客户一定信用额度的使用。（收入1，WCGW1：信用额度有3个月有效期，季度清零政策。销售内勤负责将销售信息录入销售应用系统（易飞系统）。

2、流程部副部长建立客户档案，核对客户信息，录入易飞系统，分配客户编号。（收入2，WCGW2）销售内勤定期与客户沟通，了解客户实际情况，及时更新数据库。（收入3，WCGW3）物价员根据生产基地、销售公司或进出口部提供的信息计算药品生产成本，制作"品种成本利润情况表"。物价部主管核查"品种成本利润情况表"，确保合理利润后制定药品定价方案。物价部主管审核定价方案后上报总会计师审批。（收入4，WCGW4）物价员根据审批后的方案制作"新中成药结算价格通知单"，加盖定价专用章后发送销售公司形成定价文件。

3、客户下单后，确认信用额度后，客户服务人员将销售（及新客户）录入易飞系统，生成销售订单。（收入5，WCGW5）流程部审核销售订单，根据订单在易飞系统中打印销售合同。销售内勤分类整理销售合同。（收入6，WCGW6）外勤销售人员将合同盖章后转发给经销商和流程部存档。现款销售由销售内勤直接开票。信用额度内销售由销售部经理签字；超过信用额度的销售须由经营分公司经理签字。（收入7，WCGW7）产品推广部推广的产品，每笔订单需要部门经理签字。流程部在郊区仓库设有远程开票点，负责打印出库单。

#### 二、提示说明

**提示1：关键控制点**

关键控制是指那些对于实现控制目标必不可少、与相关认定直接相关、能够有效防范和发现重大错报风险的控制。关键控制通常具有以下特征：

1. 对于实现控制目标必不可少
2. 与相关认定直接相关
3. 能够有效防范和发现舞弊或错误风险
4. 运行精确

**提示2：控制测试**

##### 2.1 缺陷类型定义

| 缺陷类型 | 定义 | 判断标准 |
|----------|------|----------|

| 严重程度 | 定义 | 判断标准 | 对财务报表的影响 |
|----------|------|----------|------------------|
| 重大缺陷 | 极有可能导致重大错报，或对财务报表产生重大影响 | 缺陷影响重大，可能导致财务报表重大错报 | 极有可能导致重大错报 |

| 调查并解决对账调节项目 | 调节项目符合预定标准将被调查和解决 | 应收账款账龄分析 |

#### 五、监督控制

| 控制类型 | 定义 | 示例 |
|----------|------|------|
| 复核其他控制的执行情况 | 观察独立验证控制实施 | 内部审计复核控制执行情况 |
| 详细复核其他控制的执行情况 | 独立详细复核控制实施，可能包括重新执行 | 应付账款主管复核调节表 |
| 定期观察控制 | 观察业务运营中控制实施或完成证据 | 盘点过程中观察销售期末盘点 |
| 定期重新执行控制 | 重新执行独立验证控制实施 | CFO复核月末结账检查表，财务总监重新执行银行账户调节表 |

#### 六、实物控制

| 控制类型 | 定义 | 示例 |
|----------|------|------|
| 仅限被授权人员与实物接触 | 实物资产或资源存储在锁定区域，仅限授权人员接触 | 支票本由应付账款经理接触，销售存储设施有门锁 |
| 定期盘点并与被审计单位记录的数量进行比较 | 定期盘点资产并与记录数量比较 | 每年库存盘点，实际数量与系统记录比较 |

#### 七、对参照数据的控制

| 控制类型 | 定义 | 示例 |
|----------|------|------|
| 匹配或录入检查 | 使用不同类型检查 | 客户主文档添加新客户时与授权客户名单匹配 |
| 限制、范围或有效性检查 | 数据录入符合预定义标准 | 价格主文档不允许负数 |
| 顺序检查 | 识别序列间隙 | 新员工ID号序列间隙 |
| 重复项目检查 | 识别重复项目 | 防止同一客户重复进入客户主文档 |
| 定期复核已设置的限额或信息的准确性 | 定期复核限额或信息准确性 | 复核供应商主文档用户名单，价格清单准确性 |
| 复核和调查有关参照数据的变更 | 变更报告列示数据变更，复核确保准确有效 | 财务总监复核工资单数据变更 |
| 已设置超过参照数据的授权限额 | 参照数据相关授权限额已设置 | 客户信用额度限制 |
| 参照数据的安全访问控制 | 参照数据输入、修改或删除仅限授权人员 | 应付账款会计员有权限添加新供应商 |

### 容易出错的领域（WCGW）示例（不归档）

| 常见容易出错的领域(WCGW) | 容易出错的领域(WCGW)的详细说明示例 | 控制活动类型 |
|-------------------------|----------------------------------|--------------|
| 交易或事件未经批准或授权 | 被审计单位将已发生并超过账单的成本确认为资产，但该合同未经授权。销售经理接受未符合认可要求的新客户。 | 授权和审批，验证 |
| 交易或事件未实际发生 | 记录了虚构的应收账款。已记录的销售信息是实际上未发货的交易 | 授权和审批，验证，对账 |
| 会计分录或其他调整未经授权 | 期末会计分录在未获授权下被过账到总账；未获授权下处理对净利润产生影响的其他调整 | 授权和审批 |
| 会计分录或其他调整不准确或与源数据不一致 | 会计分录汇总记录的工资费用与所有工资支付凭证的加总金额不符，或会计分录未记录在合适的账户或金额；冲销集团内应收账款的合并调整金额不等于集团内应收余额的加总 | 验证，对账 |
| 挪用资产 | 现金通过伪造支票被挪用；员工盗窃库存 | 授权和审批，实物控制 |
| 资产未有物理上的保护 | 库存的物理接触未限定于已授权人员，员工职责未有适当分离 | 实物控制 |
| 交易未在必要时更正 | 所有信息都不会在系统之间传输，或者传输到总帐的信息未有按照既定的常规程序或协议执行 | 验证，对账 |
| 交易处理不准确或不完整 | 员工工时输入不正确 | 验证，对账，监督控制 |
| 使用的信息或数据不可靠或不准确 | 历史索赔记录遗漏了相关信息；应收账款账龄忽略相关信息或不准确 | 验证，对账，监督控制 |
| 关键假设未得到支持 | 诉讼计提金额是在未咨询法律顾问的情况下确定的；在未查看历史索赔记录的情况下确定平均保修索赔率 | 验证，对账，监督控制 |
| 关键假设不合理或不客观 | 估计设定受益计划的养老金负债的假设工资率增长是不合理的低于历史趋势；与资产剩余使用年限有关的经济状况并不合理 | 验证，对账，监督控制 |
| 方法不符合财务报告编制基础要求 | 债券投资未按摊余成本准确记录；准备金未按照选定的会计原则在未折现的基础上确定 | 验证，对账，监督控制 |
| 计算不准确或未按所选方法来执行 | 估计的工资支出未能准确计算；资产减值计算错误地忽略了剩余价值 | 验证，对账，监督控制 |
| 方法没有一致地应用于所有计算或所有财务报告期间 | 投资和衍生品的市场价值计量不一致；应收账款坏账准备的计算未包括国外应收账款，但前期是包括的 | 验证，对账，监督控制 |
| 参照数据输入不准确 | 错误的价格输入到价格主文档中 | 验证，对参照数据的控制 |
| 参照数据不完整 | 价格主文档中未完成所有字段输入 | 验证，对参照数据的控制 |
| 参照数据未及时更新或维护 | 价格主文档未及时更新以反映价格变动 | 验证，对参照数据的控制 |
| 参照数据未获审批 | 对价格主文档的更改未被授权 | 授权和审批，对参照数据的控制 |
| 结转不准确或不完整 | 所有信息都不会在系统之间传输，或者传输到总帐的信息未有按照既定的常规程序或协议执行 | 验证，对账 |
| 交易或事件未被记录 | 被审计单位取得新的信贷额度但未有记录 | 验证，对账，监督控制 |
| 交易或事件未被及时记录 | 计提费用未包括本期的管理费用 | 验证，对账，监督控制 |
| 会计分录或其他调整不完整 | 销售系统不会为每天的销售制定常规会计分录；合并期间未记录所有公司间投资的冲销调整 | 验证 |
| 会计分录或其他调整未及时记录 | 记录工资费用的会计分录未记录在适当的期间 | 验证 |
| 资产不是由被审计单位持有或控制的 | 销售是以寄售的形式存储；被审计单位未拥有采矿权 | 授权和审批，验证 |
| 财务信息或其他信息未正确列示 | 长期债务不恰当地分类为短期债务 | 验证 |
| 财务或其他信息未得到适当描述或明确表达 | 在关联方披露中的关联方利益的描述含糊不清 | 监督控制 |
| 披露不规范 | 关联方及关联方交易存在重复或无意义的信息披露旨在掩盖对事项的正确理解 | 监督控制 |
