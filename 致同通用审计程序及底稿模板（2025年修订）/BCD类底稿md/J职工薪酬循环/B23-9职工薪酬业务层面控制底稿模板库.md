### 三、薪酬制度制定与审批

#### 3.1 职工薪酬制度制定审批流程图

| 人力资源部 | 薪酬委员会 | 总经理 | 董事会 | 会计 |
|------------|------------|--------|--------|------|
| 薪酬方案制定 | 专业评估审查 | 管理层审批 | 重大决策审批 | 核算记录 |

#### 3.2 薪酬制定流程详细说明

**开始** → **薪酬市场调研** → **制定薪酬方案** → **薪酬委员会审查** → **总经理审批** → **董事会决策** → **制度发布执行**

**流程说明 - A**
人力资源部门启动薪酬制度制定工作，进行市场薪酬调研，分析同行业薪酬水平，结合公司实际情况制定薪酬方案，确保薪酬制度的科学性和合理性。

**流程说明 - B**
薪酬委员会对薪酬方案进行专业评估和审查，从薪酬结构、薪酬水平、激励机制等方面进行全面评价，确保方案的可行性。

**流程说明 - C**
总经理对薪酬方案进行管理层审批，董事会对重大薪酬政策进行最终决策，确保薪酬制度符合公司战略和治理要求。

#### 3.3 容易出错的领域(WCGW)及控制措施

| 控制点编号 | WCGW(容易出错的领域) | WCGW(容易出错的领域)详细描述 | 潜在控制 |
|------------|---------------------|---------------------------|----------|
| 薪酬1 | 制度制定不科学 | 1. 薪酬制度制定缺乏市场调研和科学论证，导致制度不合理 | 薪酬制度制定必须进行市场调研，形成详细的薪酬方案报告，经薪酬委员会审查 |
| 薪酬2 | 审批程序不完整 | 2. 薪酬制度未经过完整的审批程序，或审批权限不明确 | 建立明确的薪酬制度审批程序，重大薪酬政策须经董事会审批 |

| 在确定是否识别出一项或多项控制缺陷时，项目组需要运用职业判断，考虑的因素包括：<br>• 缺失或无效的控制（包括 IT 一般控制），包括应对容易出错的领域的控制；<br>• 职责分离问题（称为不相容职责分离） ；<br>• 控制未经适当的有胜任能力的人员执行；<br>• 控制运行的频率或时间不恰当；<br>• 控制不足够精确；<br>• 不适当的控制组合，考虑：<br>- 检查性控制和预防性控制；<br>- 无正式记录的控制；<br>- 监督或监控性控制和交易性控制。 |  |  |  |  |  |  |
