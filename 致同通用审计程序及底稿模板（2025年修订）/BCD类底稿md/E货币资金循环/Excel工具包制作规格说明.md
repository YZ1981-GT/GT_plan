## 🛠️ 工具1：余额调节表核对表

### 文件基本信息
- **文件名**: `01-余额调节表核对表.xlsx`
- **工作表数量**: 3个
- **主色调**: 蓝色系（专业审计色）
- **保护**: 公式单元格锁定，数据单元格可编辑

### 工作表结构

#### Sheet1: 基础信息
**用途**: 录入账户基本信息

**布局**:
```
A1:F1 - 标题行："货币资金银行余额调节表核对工具"
A2:F2 - 项目信息：被审计单位、审计期间、编制人

A4:B4 - 账户信息标题
A5 - 银行名称
A6 - 账号
A7 - 币种
A8 - 对账日期

C5:D5 - 输入框（下拉或手动）
```

**字段说明**:
| 字段 | 类型 | 验证规则 | 示例 |
|------|------|---------|------|
| 被审计单位 | 文本 | 必填 | ABC公司 |
| 审计期间 | 日期 | 必填，格式YYYY-MM-DD | 2024-12-31 |
| 编制人 | 文本 | 必填 | 张三 |
| 银行名称 | 文本 | 必填 | 工商银行XX支行 |
| 账号 | 文本 | 19位 | 1234567812345678901 |
| 币种 | 下拉 | RMB/USD/EUR | RMB |
| 对账日期 | 日期 | 必填 | 2024-12-31 |

---

#### Sheet2: 调节计算
**用途**: 核心计算表，自动计算调节后余额

**布局**:
```
第1部分：银行对账单部分（A列）
A3: 银行对账单余额
A4: 加：企业已收银行未收
A5-A14: [明细行1-10]
A15: 小计
A16: 减：企业已付银行未付
A17-A26: [明细行1-10]
A27: 小计
A28: 调节后余额（A）

第2部分：企业账面部分（C列）
C3: 企业账面余额
C4: 加：银行已收企业未收
C5-C14: [明细行1-10]
C15: 小计
C16: 减：银行已付企业未付
C17-C26: [明细行1-10]
C27: 小计
C28: 调节后余额（B）

第3部分：差异分析（E列）
E3: 差异（A-B）
E4: 差异原因
E5: 差异性质（正常/异常）
```

**公式**:
```excel
// A15单元格（加项小计）
=SUM(A5:A14)

// A27单元格（减项小计）
=SUM(A17:A26)

// A28单元格（调节后余额A）
=A3+A15-A27

// C28单元格（调节后余额B）
=C3+C15-C27

// E3单元格（差异）
=A28-C28

// E5单元格（差异性质判断）
=IF(ABS(E3)<0.01,"相符",IF(E3=0,"相符","不符⚠️"))
```

**条件格式**:
- E3单元格：如果<>0，背景色改为红色，字体白色
- E5单元格：如果包含"不符"，背景色改为红色
- A28和C28：如果不相等，边框加粗红色

**数据验证**:
- A3, C3: 数值型，允许负数，保留2位小数
- A5:A14, A17:A26, C5:C14, C17:C26: 数值型，允许0，保留2位小数
- 金额字段格式：#,##0.00

---

#### Sheet3: 检查清单
**用途**: 自动生成检查清单

**布局**:
```
A1: 标题"银行余额调节表检查清单"
A3: 序号 | B3: 检查项目 | C3: 检查结果 | D3: 说明

检查项：
1. 调节表格式是否规范
2. 调节后余额是否相符
3. 未达账项是否有支持文件
4. 是否有长期未达账项（>3个月）
5. 编制人、复核人是否签字
6. 余额是否与函证一致
7. 余额是否与审定表一致
8. 期后验证是否完成
```

**自动填充逻辑**:
- C列根据Sheet2的计算结果自动填充"通过"/"未通过"
- 差异=0 → "通过"，否则"未通过⚠️"

**打印设置**:
- 页面方向：纵向
- 纸张：A4
- 页边距：常规
- 页眉：被审计单位名称 + 审计期间
- 页脚：编制人 + 日期

## 🛠️ 工具2：大额交易筛选表

### 文件基本信息
- **文件名**: `02-大额交易筛选表.xlsx`
- **工作表数量**: 4个
- **主色调**: 绿色系
- **特点**: 支持导入银行流水CSV

### 工作表结构

#### Sheet1: 参数设置
**用途**: 设置筛选参数

**字段**:
```
重要性水平: [___] 万元
大额交易阈值: [___] 万元（默认=重要性水平×50%）
筛选日期范围: [起始日期] 至 [结束日期]
交易类型: □收入 □支出 □全部
```

---

#### Sheet2: 原始数据导入
**用途**: 粘贴银行流水数据

**列结构**:
```
A: 日期
B: 摘要/对方账户名称
C: 收入金额
D: 支出金额
E: 余额
F: 对方账号（可选）
G: 附注
```

**数据验证**:
- 日期：日期型
- 金额：数值型，≥0

---

#### Sheet3: 筛选结果
**用途**: 自动筛选大额交易

**筛选条件**（使用高级筛选或公式）:
```excel
// 收入大额
=C2>=参数设置!$B$4*10000

// 支出大额
=D2>=参数设置!$B$4*10000
```

**增加列**:
```
H列: 交易类型（收入/支出）
I列: 交易金额（C+D中的非零值）
J列: 异常标识
K列: 检查状态
```

**异常标识规则**（自动判断）:
```excel
// 快进快出（3天内）
=IF(AND(前后3天内有相同金额的反向交易),"快进快出⚠️","")

// 个人往来
=IF(ISNUMBER(SEARCH("个人",B2)),"个人往来⚠️","")

// 节假日交易
=IF(WEEKDAY(A2)=1,OR(WEEKDAY(A2)=7),"节假日交易⚠️","")

// 整数金额
=IF(MOD(I2,100000)=0,"整数金额⚠️","")
```

**条件格式**:
- J列如果不为空：背景黄色，字体红色

---

#### Sheet4: 汇总分析
**用途**: 统计大额交易情况

**内容**:
```
1. 大额收入笔数、金额
2. 大额支出笔数、金额
3. 异常交易笔数、占比
4. 前10大收入交易
5. 前10大支出交易
6. 每月大额交易趋势图（折线图）
```

**图表**:
- 类型：折线图
- X轴：月份
- Y轴：交易笔数和金额（双轴）

## 🛠️ 工具3：函证跟踪表

### 文件基本信息
- **文件名**: `03-函证跟踪表.xlsx`
- **工作表数量**: 3个
- **主色调**: 橙色系
- **特点**: 自动提醒催函

### 工作表结构

#### Sheet1: 函证登记
**用途**: 登记所有函证发出情况

**列结构**:
```
A: 序号
B: 银行名称
C: 账号（后4位）
D: 期末余额（元）
E: 函证编号
F: 函证日期
G: 寄出日期
H: 快递单号
I: 预计回函日期（寄出日期+10天）
J: 实际回函日期
K: 回函状态（未回/已回/差异）
L: 回函余额（元）
M: 差异金额
N: 催函次数
O: 备注
```

**公式**:
```excel
// I列（预计回函日期）
=G2+10

// K列（回函状态）
=IF(J2="","未回",IF(M2=0,"已回-相符","已回-差异⚠️"))

// M列（差异金额）
=L2-D2

// N列（催函次数）
=COUNTIF(催函记录!A:A,A2)
```

**条件格式**:
- K列"未回" + TODAY()>I列：背景红色（逾期未回）
- K列"已回-差异"：背景黄色
- M列≠0：字体红色加粗

---

#### Sheet2: 催函记录
**用途**: 记录催函情况

**列结构**:
```
A: 函证序号（关联Sheet1）
B: 催函日期
C: 催函方式（电话/邮件/现场）
D: 联系人
E: 对方反馈
F: 承诺回函日期
G: 执行人
```

---

#### Sheet3: 函证汇总
**用途**: 汇总函证情况

**内容**:
```
1. 函证总数
2. 已回函数量、比例
3. 未回函数量、列表
4. 差异函证数量、列表
5. 平均回函时间
6. 函证进度饼图
```

**提醒功能**:
在Sheet1顶部增加提醒区域：
```
⚠️ 逾期未回函证：[数量]个
⚠️ 存在差异函证：[数量]个
📅 今日需催函：[数量]个
```

## 🛠️ 工具4：小金库检查清单

### 文件基本信息
- **文件名**: `04-小金库检查清单.xlsx`
- **工作表数量**: 4个
- **主色调**: 紫色系
- **特点**: 风险评分机制

### 工作表结构

#### Sheet1: 8大预警信号
**用途**: 逐项检查预警信号

**检查项**:
```
序号 | 预警信号 | 检查结果 | 证据 | 风险分值

1. 征信报告账户数 > 账面账户数
2. 新开户审批文件不齐
3. 大额现金存入，来源不明
4. 频繁现金取款，用途不明
5. 客户反映已付款，但账面未收到
6. 发票开具数量 > 账面收入确认数量
7. 资金转入个人账户后无合理解释
8. 应收账款长期挂账，客户已付款
```

**检查结果下拉**:
- 未发现（0分）
- 疑似（5分）
- 确认（10分）

**总分计算**:
```excel
=SUM(风险分值列)
```

**风险评级**:
```
0-15分：低风险
16-40分：中风险⚠️
>40分：高风险🚨
```

---

#### Sheet2: 6大排查方法
**用途**: 记录排查过程

**检查项**:
```
1. 征信报告核对

   - 账面账户数量
   - 差异账户列表
   
2. 函证交叉核对
   - 函证回函情况
   - 未发现账户
   
3. 网银现场检查
   - 检查日期
   - 检查人
   - 网银显示账户数
   - 发现情况
   
4. 大额现金存取分析
   - 筛选标准
   - 可疑交易数
   
5. 个人账户交易检查
   - 筛选标准
   - 可疑交易数
   
6. 管理层访谈
   - 访谈日期
   - 访谈对象
   - 访谈内容
   - 访谈结论
```

---

#### Sheet3: 征信vs账面对比
**用途**: 详细对比征信报告和账面账户

**列结构**:
```
A: 序号
B: 开户行
C: 账号（后4位）
D: 征信报告（有/无）
E: 账面记录（有/无）
F: 一致性（一致/不一致⚠️）
G: 差异说明
```

**自动判断**:
```excel
// F列（一致性）
=IF(AND(D2="有",E2="有"),"一致",

```

**条件格式**:
- F列"不一致"：整行背景黄色

## 🛠️ 工具5：期后流水分析表

### 文件基本信息
- **文件名**: `05-期后流水分析表.xlsx`
- **工作表数量**: 3个
- **主色调**: 青色系
- **特点**: 自动识别异常模式

### 工作表结构

#### Sheet1: 期后流水导入
**用途**: 粘贴期后1-2个月的银行流水

**列结构**（同工具2-Sheet2）:
```
A: 日期
B: 摘要
C: 收入
D: 支出
E: 余额
F: 对方账号
```

**标识列**（自动计算）:
```
G: 交易类型（期初余额核对/未达账项核实/正常交易/异常交易）
H: 异常类型
I: 检查状态
```

---

#### Sheet2: 异常识别
**用途**: 自动识别4类异常

**识别规则**:
```
1. 期初余额差异
   - 检查期后第一笔余额是否=12/31余额
   
2. 大额转出（>50万）
   - 筛选支出>50万的交易
   - 标识对方为个人/非经营单位
   
3. 快速进出（<3天）
   - 检测3天内金额相近的一进一出
   
4. 频繁关联方往来
   - 检测与关联方的频繁交易（>5笔/月）
```

**自动标记**:
```excel
// G列（交易类型）
=IF(ROW()=2,"期初余额核对",
   IF(ISNUMBER(MATCH(B2,未达账项列表)),"未达账项核实",
     IF(OR(D2>500000,快速进出,关联方频繁),"异常交易⚠️","正常交易")))
```

---

#### Sheet3: 检查报告
**用途**: 生成期后流水检查报告

**内容**:
```
一、期初余额核对
   - 12/31期末余额：[___]
   - 期后第一天余额：[___]
   - 差异：[___]（✓相符/✗不符）
   
二、未达账项核实
   - 未达账项总数：[___]
   - 已核实：[___]
   - 未核实：[___]
   - 未核实明细：[列表]
   
三、异常交易发现
   - 大额转出：[___]笔，金额[___]万
   - 快速进出：[___]笔
   - 关联方频繁：[___]笔
   - 详细清单：[列表]
   
四、审计结论
   - □ 期后流水正常，无异常
   - □ 发现异常，需进一步调查
   
五、后续程序
   [文本框]
```

## 🎨 通用设计规范

### 颜色方案
```
标题行：深蓝色背景（RGB: 0,32,96），白色字体
子标题：浅蓝色背景（RGB: 68,114,196），白色字体
数据区域：白色背景，黑色字体
合计行：浅灰色背景（RGB: 217,217,217）
警告/错误：红色背景（RGB: 255,0,0），白色字体
提醒/注意：黄色背景（RGB: 255,255,0），黑色字体
通过/正常：绿色字体（RGB: 0,176,80）
```

### 字体规范
```
标题：微软雅黑，14pt，加粗
子标题：微软雅黑，12pt，加粗
正文：宋体，11pt，常规
备注：宋体，9pt，斜体
```

### 单元格格式
```
日期：yyyy-mm-dd
金额：#,##0.00（千位分隔符，保留2位小数）
百分比：0.00%（保留2位小数）
文本：左对齐
数值：右对齐
标题：居中对齐
```

### 数据验证
- 所有输入字段必须有数据验证
- 日期字段：限制为日期型
- 金额字段：限制为数值型，≥0
- 下拉字段：提供选项列表
- 必填字段：使用条件格式标识（空=红色边框）

### 保护设置
- 锁定公式单元格
- 锁定格式
- 允许筛选和排序
- 允许插入/删除行（在数据区域）

### 打印设置
- 页眉：被审计单位名称 | 审计期间
- 页脚：编制人：___ | 日期：___ | 第 X 页，共 Y 页
- 页边距：上下2cm，左右1.5cm
- 重复标题行：第1-2行
