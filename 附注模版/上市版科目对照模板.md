# 上市版 — 报表科目与附注对照模板

> 本文档列示上市公司报表附注中"财务报表主要项目注释"章节下的科目结构。
> 每个表格标注了校验角色，规则引擎直接按标注执行，无需过滤规则。
>
> 校验角色说明：
> - `[余额]` — 参与报表 vs 附注余额核对（仅取合计行 vs 报表，表格内无需任何计算）
> - `[宽表]` — 参与横向公式校验（movement 或 category_sum），合计行同时用于余额核对
> - `[交叉]` — 参与同科目表间交叉校验或跨科目交叉校验
> - `[其中项]` — 参与其中项校验（明细行之和 = 合计行），合计行同时用于余额核对
> - `[描述]` — 纯描述性/政策性表格，不参与数值校验
>
> **互斥规则：`[余额]` 不与 `[其中项]`/`[宽表]` 共存。**
> 含其中项加总或宽表横向公式的表格，余额核对从计算结果的合计行自然得出，
> 无需单独标注 `[余额]`。`[余额]` 仅用于"纯取合计行比报表"的简单表格。
>
> 与国企版结构一致的科目仅列出差异，其余参照国企版。

## 一、合并财务报表主要项目注释

### 1. 资产负债表科目

> 以下科目与国企版结构一致，校验角色相同（不再重复列出）：
> 货币资金、交易性金融资产、衍生金融资产、应收票据、应收账款、应收款项融资、
> 预付款项、其他应收款、存货、合同资产、持有待售资产、一年内到期的非流动资产、
> 其他流动资产、债权投资、其他债权投资、长期应收款、长期股权投资、
> 其他权益工具投资、其他非流动金融资产、投资性房地产、
> 在建工程、生产性生物资产、油气资产、开发支出、商誉、长期待摊费用、
> 递延所得税资产/负债、其他非流动资产、
> 短期借款、交易性金融负债、衍生金融负债、应付票据、应付账款、预收款项、
> 合同负债、应付职工薪酬、应交税费、其他应付款、持有待售负债、
> 其他流动负债、长期借款、应付债券、租赁负债、长期应付款、
> 预计负债、递延收益、其他非流动负债、
> 其他权益工具、资本公积、专项储备、盈余公积、未分配利润

#### 以下为上市版差异科目：

## 三、上市版与国企版差异汇总

| 差异项 | 上市版 | 国企版 |
|--------|--------|--------|
| 固定资产/无形资产/使用权资产 | category_sum（分类合计型） | movement（变动公式型） |
| 母公司附注多"应收票据" | 有 | 无 |
| 设定受益计划净资产 | 有 | 无 |
| 库存股 | 有 | 无 |
| 股本 vs 实收资本 | 股本 | 实收资本 |
| 母公司附注多"现金流量表补充资料" | 无 | 有 |
| 一年内到期的非流动负债子项 | 含"一年内到期的租赁负债" | 含"一年内到期的长期应付款" |
| 其他综合收益 | 资产负债表中单独列示 | 仅在利润表附注中 |
