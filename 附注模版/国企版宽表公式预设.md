# 国企版 — 宽表横向公式预设

> 本文档列示国企报表附注中常见宽表（多列变动表）的标准结构和横向校验公式。
> 系统优先使用预设公式进行规则匹配，匹配失败时再调用 LLM 分析。
> 国企版宽表以 **movement（变动公式型）** 为主：行=资产类别，列=期初/变动项/期末。

---

## 一、长期股权投资

### 1. 长期股权投资分类

| 列 | 角色 | 符号 |
|----|------|------|
| 项目 | label | — |
| 期初余额 | opening | + |
| 本期增加 | movement | + |
| 本期减少 | movement | - |
| 期末余额 | closing | = |

公式：`期初余额 + 本期增加 - 本期减少 = 期末余额`

### 2. 长期股权投资明细（对联营/合营/子公司）

| 列 | 角色 | 符号 |
|----|------|------|
| 被投资单位 | label | — |
| 投资成本 | skip | — |
| 期初余额（账面价值） | opening | + |
| 减值准备期初余额 | skip | — |
| 追加/新增投资 | movement | + |
| 减少投资 | movement | - |
| 权益法下确认的投资损益 | movement | + |
| 其他综合收益调整 | movement | + |
| 其他权益变动 | movement | + |
| 宣告发放现金股利或利润 | movement | - |
| 计提减值准备 | movement | - |
| 其他 | movement | + |
| 期末余额（账面价值） | closing | = |
| 减值准备期末余额 | skip | — |

公式：`期初账面价值 + 追加投资 + 投资损益 + 其他综合收益 + 其他权益变动 - 减少投资 - 现金股利 - 计提减值 + 其他 = 期末账面价值`

注意：投资成本列和减值准备列不参与横向公式。

## 三、固定资产（多段变动表）

国企版固定资产为多段表格，每段独立适用横向公式：
- 一、账面原值：期初 + 增加 - 减少 = 期末
- 二、累计折旧：期初 + 增加 - 减少 = 期末
- 三、账面净值：不适用横向公式（纵向计算：原值 - 折旧）
- 四、减值准备：期初 + 增加 - 减少 = 期末
- 五、账面价值：不适用横向公式（纵向计算：原值 - 折旧 - 减值）

| 列 | 角色 | 符号 |
|----|------|------|
| 项目/类别 | label | — |
| 期初余额 | opening | + |
| 本期增加 | movement | + |
| 本期减少 | movement | - |
| 期末余额 | closing | = |

公式：`期初余额 + 本期增加 - 本期减少 = 期末余额`（每段独立）
