# 上市版 — 宽表横向公式预设

> 本文档列示上市公司报表附注中常见宽表的标准结构和横向校验公式。
> 上市版与国企版的核心差异：固定资产/无形资产/使用权资产等采用 **category_sum（分类合计型）** 布局。
> 其余科目（长期股权投资、开发支出、商誉等）与国企版结构基本一致。

---

## 一、长期股权投资明细

与国企版结构一致。

| 列 | 角色 | 符号 |
|----|------|------|
| 被投资单位 | label | — |
| 期初余额（账面价值） | opening | + |
| 减值准备期初余额 | skip | — |
| 追加/新增投资 | movement | + |
| 减少投资 | movement | - |
| 权益法下确认的投资损益 | movement | + |
| 其他综合收益调整 | movement | + |
| 其他权益变动 | movement | + |
| 宣告发放现金股利或利润 | movement | - |
| 计提减值准备 | movement | - |
| 其他 | movement | + |
| 期末余额（账面价值） | closing | = |
| 减值准备期末余额 | skip | — |

公式：`期初账面价值 + 追加投资 + 投资损益 + 其他综合收益 + 其他权益变动 - 减少投资 - 现金股利 - 计提减值 + 其他 = 期末账面价值`

---

## 二、固定资产（分类合计型 category_sum）

上市版固定资产为分类合计型布局：
- 行 = 变动项目（期初余额、本期增加、本期减少、期末余额等）
- 列 = 资产类别（房屋及建筑物、机器设备、运输设备、…、合计）
- 多段结构：一、账面原值 → 二、累计折旧 → 三、减值准备 → 四、账面价值

| 列 | 角色 | 符号 |
|----|------|------|
| 项目 | label | — |
| 房屋及建筑物 | data | + |
| 机器设备 | data | + |
| 运输设备 | data | + |
| …（其他类别） | data | + |
| 合计 | total | = |

公式：`房屋及建筑物 + 机器设备 + 运输设备 + … = 合计`

注意：由 `_try_build_category_sum_from_headers` 自动检测，无需手动预设。

## 五、在建工程项目变动

与国企版结构一致。

公式：`期初余额 + 本期增加 - 转入固定资产 - 其他减少 = 期末余额`

---

## 六、开发支出

与国企版结构一致。

公式：`期初余额 + 内部开发支出 + 其他增加 - 确认为无形资产 - 计入当期损益 - 其他减少 = 期末余额`

---

## 十七、权益类科目

以下科目均为标准 `期初 + 增加 - 减少 = 期末` 结构：
- 股本（对应国企版的"实收资本"）
- 资本公积
- 库存股（上市版特有）
- 专项储备
- 盈余公积

---

## 附：上市版与国企版的主要差异

| 科目 | 上市版 | 国企版 |
|------|--------|--------|
| 固定资产 | category_sum（行=变动项，列=类别+合计） | movement（行=类别，列=期初/增减/期末） |
| 无形资产 | category_sum（同上） | movement（同上） |
| 使用权资产 | category_sum（同上） | movement（同上） |
| 股本 vs 实收资本 | 股本 | 实收资本 |
| 库存股 | 有 | 无 |
| 设定受益计划净资产 | 有 | 无 |
| 商誉增减列 | 可扩展（"可无限量添加列"） | 固定列 |
| 持有待售资产减值 | 增加/转回/出售 | 增加/减少 |
